Privacy Policy for tiktokbot-content

Last updated: 29. 12. 2025

1. Overview tiktokbot-content is a private automation tool developed for personal use to manage and upload educational content to TikTok.

2. Data Collection and Usage This application does not collect, store, or process personal data from any third-party TikTok users. The application only accesses the developer's own account information via the TikTok API for the purpose of content distribution.

3. Data Sharing No data is shared with third parties, except for the necessary communication with TikTok's official APIs to perform its intended functions.

4. Data Security All API keys and access tokens used by tiktokbot-content are stored locally and securely by the developer and are never exposed to public access.

5. Contact Information For any inquiries regarding this policy, please contact the developer through the GitHub repository issues or profile.
